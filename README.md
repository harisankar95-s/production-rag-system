# AI Research Assistant

A production-grade multi-agent RAG system built with LangGraph, ChromaDB, and Gemini.

## Features
- Hybrid search (BM25 + vector search)
- Cross-encoder reranking
- Multi-agent routing (RAG + web search)
- RAGAS evaluation metrics
